gpsNav

The application for online GPS tracking

gpsNav Application - GitHub Pages

Welcome to the gh-pages branch of my gpsNav application! Below are two basic demos of how you could use this application as an online GPS tracking device - and most importantly, how you could customize it according to your own wants and needs.

To initialize the gpsNav app, just simply call gpsNav.init(); after you have made a link to the gpsNav.min.js (or gpsNav.js) file.

gpsNav.init();

Try Me

To change the settings of the application, simply pass an object as the argument for the init() function.

gpsNav.init({
    zoom: 17,
    width: 800,
    height: 600,
    error: {
        permissionDenied: {
            image: "permissionDenied.jpg",
            text: "Come on, I won't use your information!"
        },
        positionUnavailable: {
            image: "positionUnavailable.jpg",
            text: "Oh, that sucks!  Are you lost?"
        },
        timeout: {
            image: "timeout.gif",
            text: "Oh, I'm really sorry.  It seems your position has timed out."
        },
        unknownError: {
            image: "unknownError.png",
            text: "Sorry, I don't know what happened to the app or your computer.  Chances are that the problem is in your computer (haha!*).  *Sorry, that was a bit rude."
        }
    }
});

Try Me

You could also try the following settings.

gpsNav.init({
    zoom: 19,
    width: 400,
    height: 400,
    error: {
        permissionDenied: {
            image: "permissionDenied.jpg",
            text: "Sorry, you denied permission for the website to use your location."
        },
        positionUnavailable: {
            image: "positionUnavailable.jpg",
            text: "Sorry, your position is currently unavailable."
        },
        timeout: {
            image: "timeout.gif",
            text: "Sorry, your position has timed out.  Please try refreshing the page."
        },
        unknownError: {
            image: "unknownError.png",
            text: "Sorry, an unknown error occurred."
        }
    }
});
