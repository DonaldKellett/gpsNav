gpsNav App Demo Page

This page is to show how the app can be used. Before using the app, the following HTML must be pasted somewhere on your webpage/app:

<img src="placeholder.jpg" id="map" />
<p><span id="error"></span></p>

The following link to the script must also be pasted:

<script src="gpsNav.js"></script>

To initialize/activate the app, simply paste this script after gpsNav.js:

<script>gpsNav.init();</script>

If you want to add your own custom settings (such as changing the size of the map), please read README.md to learn more.
